While the core of Lunatech's work is large applications based on open-source Java EE stacks, usually with a web-based user-interface, we are increasingly involved in projects to build pure web applications. At the same time, web standards such as HTML5 allow for increasingly-sophisticated web application user-interfaces, demanding correspondingly sophisticated design and front-end development skills.

These applications focus more on user-interaction and visual design, and delegate data processing and business logic to other server-side systems, via web services and shared databases.

To help us build these kinds of applications, we are looking for a front-end developer who is an expert in:

  • JavaScript
  • jQuery
  • HTML5
  • CSS 3.

We are looking for someone who is a programmer first and a designer second, because the work is more about jQuery and HTTP than it is about Photoshop and graphic design. Although Java programming experience is not required, you will be working with Java-based server technologies. The ideal candidate is:

  • passionate about the web
  • up-to-date in current issues in web design
  • creative and able to produce both derivative and innovative
  • able to explain front-end design to back-end systems developers
  • following the people currently on the cutting-edge of web standards development.

All staff take on delivery responsibility, and our senior developers manage our development projects, instead of non-coding project managers.

In addition, we are looking for a useful combination of skills and experience with things like:

  • Ajax and JSON
  • user-interaction design
  • user-centred design and testing
  • web-based usability and accessibility
  • mobile application design
  • graphic design
  • web development frameworks such as PHP, Ruby on Rails, Grails or the Play framework
  • comfortable working in OS X or Linux.
